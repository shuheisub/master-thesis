# master-thesis


## 説明
* 修士論文で使用したスクリプト

## 起動方法
* Anacondaを使用する場合
```
# install pakages
$ conda install --file requirements.txt 

$ python -m app
```

* pipを使用する場合
```
# install pakages
$ pip install -r requirements.txt

$ python -m app
```
